System dynamics

Sphero Robot Model

For System Dynamics, my group selected a Sphero robot as a compact electromechanical system to model. We derived the governing ODEs, converted them into transfer functions, and used block diagrams and bode plots to describe the system response.

The project was useful because it forced the physical robot to become an analytical model: assumptions had to be explicit, variables had to connect to motion, and the final plots had to explain behavior rather than stand alone.

Technologies used: MATLAB, system dynamics, transfer functions, bode plots
